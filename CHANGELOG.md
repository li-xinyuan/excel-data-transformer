# 更新日志

## [1.9.2] - 2026-03-23

### 字段映射置信度规则优化
- **精确匹配 100 分**：字段名称完全一致（去掉 `*` 并 trim 后比较）时返回 100 分高置信度，显示绿色连线
- **规则定义**：
  - 🔵 蓝色 = 手动映射（matchType === 'manual'）
  - 🟢 绿色（高置信度）= score = 100
  - 🟠 橙色（中置信度）= score >= 80
  - 🔴 红色（低置信度）= score < 80

### 数据转换预览界面优化
- **连线起点调整**：连线起点从源数据表头改为源数据行的对应单元格下边缘，终点保持在目标数据表头上边缘，避免与数据行混杂
- **起点终点圆点**：连线起点和终点都显示小圆点标记
- **行号选择器优化**：移除 number input 内置的上下箭头（spin button），只保留外部的 ▲▼ 按钮
- **预览行数联动**："预览行数"显示当前选中行号（如 "3 / 100"），随行号选择器变化联动更新

### 值转换规则功能完善
- **转换规则标记**：有值转换规则的映射连线中间显示 ⚙ 图标标记
- **单元格背景提示**：目标数据单元格有转换规则时显示黄色背景
- **转换结果正确应用**：修复"开始转换"时值转换规则未正确应用的问题
- **统一转换方法**：预览接口和转换接口共用同一套 `applyValueTransformForOutput` 函数，确保所有转换类型（simple、string、date、number、logic）处理一致

### 代码优化
- **公共函数提取**：将 `applyValueTransformForOutput` 提取为公共函数，统一值转换规则的处理逻辑
- **转换接口修复**：在 server.js 转换接口中正确附加值转换规则到 columnMappings

## [1.9.0] - 2026-03-22

### 数据预览功能增强
- **实时预览模式**：新增"👀 预览转换结果"按钮，在正式转换前可查看转换效果
- **对比视图**：源数据和目标数据上下排列，清晰展示转换前后对比
- **SVG 连线展示**：使用贝塞尔曲线连接源字段和目标字段，直观显示映射关系
- **颜色区分映射**：根据映射置信度使用不同颜色（绿色 - 高、橙色 - 中、红色 - 低、蓝色 - 手动）
- **单行数据预览**：默认显示第 1 行数据，支持切换查看不同行的转换效果
- **行号控制**：源数据左侧显示行号输入框，支持按钮切换和手动输入行号
- **同步显示**：修改行号时源数据和目标数据同步切换，体现一对一映射关系
- **值转换规则应用**：预览时自动应用已配置的值转换规则（如 substring、replace、trim 等）
- **统计信息展示**：显示总行数、预览行数、已映射字段数等统计信息
- **界面紧凑优化**：表格字段竖向排列，节省空间，一屏显示更多列

### 支持的转换操作
- **字符串操作**：substring（截取）、replace（替换）、trim（去空格）、uppercase（转大写）、lowercase（转小写）
- **数字操作**：round（四舍五入）、floor（向下取整）、ceil（向上取整）、fixed（保留小数）

### 界面优化
- **按钮布局调整**：保存按钮移至预览和转换按钮之间，操作流程更顺畅（预览 → 保存 → 转换）
- **标题移除**：预览弹窗去掉多余标签页，直接显示对比视图
- **弹窗宽度增加**：从 1000px 增加到 1400px，可显示更多字段
- **数据区域优化**：预览数据区域高度增加，展示更多内容

### 问题修复
- **预览数据星号显示**：目标字段名称保留必填标记星号（*），不再移除
- **emoji 图标修复**：预览弹窗标题 emoji 图标正确显示
- **提示图标风格统一**：蓝色和红色圆点图标使用卡通风格（👀），与实际连接点风格一致
- **闪烁效果统一**：提示圆点添加闪烁动画，与实际连接点脉冲效果相似

## [1.8.0] - 2026-03-21

### 配置管理功能
- **独立配置管理界面**：新增右上角"⚙️ 配置管理"按钮，无需上传文件即可管理配置
- **选择性导出配置**：支持勾选单个或多个配置进行导出，导出文件包含完整配置信息
- **批量导入配置**：支持导入单个或批量配置文件，智能处理配置冲突
- **冲突处理机制**：导入同名配置时提示选择"覆盖"或"跳过"，避免误操作
- **配置列表详情**：配置管理弹窗中显示配置创建时间、源字段数、目标字段数、映射数等详细信息
- **删除配置功能**：支持在配置管理界面删除配置，删除前弹出确认提示
- **跨浏览器/设备支持**：导出的 JSON 配置文件可在任何浏览器或电脑上导入使用
- **配置按钮精简**：移除配置选择器旁的导入、导出、删除按钮，统一在配置管理界面操作

### 用户体验优化
- **控制面板可拖拽**：底部控制面板新增拖拽手柄，可拖动到任意位置，避免遮挡字段映射
- **位置记忆功能**：控制面板拖动位置自动保存，刷新页面后恢复到上次位置
- **拖拽提示优化**：拖拽手柄显示"🖱️ 点击可拖动控制面板"文字提示
- **配置信息栏布局优化**：配置信息栏移至配置选择器和保存按钮之间，布局更合理
- **控制面板紧凑化**：优化控制面板宽度，自适应内容大小，减少空白区域
- **布局居中对齐**：配置行内容居中对齐，视觉效果更美观

### 界面美化
- **手动映射提示优化**：提示文字分行显示，每句话独立一行，更易阅读
- **图标化提示**：使用蓝色●、红色●、绿色──等图标替代文字描述，更直观
- **第三步闪烁提示**：当有必填字段未映射时，手动映射提示框边缘橙色闪烁，吸引注意
- **舒适闪烁频率**：闪烁频率调整为 3 秒/次，光晕柔和，既醒目又不焦虑
- **颜色突出显示**：提示文字中的"蓝色"、"变红"等关键词使用对应颜色加粗显示

### 问题修复
- **按钮空值错误修复**：修复设置按钮 disabled 属性时的空值检查，避免"Cannot set properties of null"错误
- **保存配置弹窗修复**：修复保存配置确认对话框的异步处理逻辑，确保弹窗正确关闭
- **配置加载优化**：优化配置加载流程，确保配置正确应用并更新 UI 状态

- **配置列表自动刷新**：保存配置后自动刷新配置下拉列表并选中刚保存的配置

- **转换时参数错误修复**：修复 `buildOutputRows` 和 `writeOutputFile` 函数参数顺序错误
- **Workbook 无效错误修复**：修复 `writeOutputFile` 函数缺少返回值导致的"Invalid Workbook"错误
- **必填字段检测优化**：点击"开始转换"时重新计算必填字段映射情况，确保提示准确
- **配置加载空值检查**：添加配置加载时的空值检查，避免加载不存在的配置时报错
