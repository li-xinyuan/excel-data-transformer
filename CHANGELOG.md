# 更新日志

## [1.6.0] - 2026-03-19

### 新增功能
- **连线交互优化**：连线现在显示在面板上方，起点和终点不会被遮挡
- **操作提示更新**：更新手动映射提示文字，包含建立映射、删除映射和设置转换规则的完整说明

### 界面优化
- **连线图层调整**：调整 SVG canvas 的 z-index，确保连线始终显示在最上层
- **转换规则指示器清除**：删除映射或删除文件时，自动清除相关的转换规则指示器

### 问题修复
- **必填缺失统计修复**：修复必填缺失字段数量计算逻辑，现在会正确统计已映射的必填字段
- **服务持续运行**：移除生成文件后自动关闭服务的逻辑，服务现在会持续运行
- **连线点击区域修复**：修复连线点击区域问题，现在可以在连线任何位置点击删除
- **⚙按钮点击修复**：修复连线中间设置按钮点击无反应的问题
- **文件删除按钮修复**：修复文件删除按钮被遮挡无法点击的问题

### 字段匹配逻辑优化
- **高置信度**：字段名称完全一样时（去掉 `*` 并 trim 后比较）
- **中置信度**：字段里包含相同含义的词语
- **无匹配**：没有共同词语时不再建立低置信度映射

---

## [1.5.0] - 2025-03-16

### 新增功能
- **数据预览弹出框**：点击源文件行数可弹出预览窗口，显示前5行数据
- **配置变化检测**：转换前自动检测配置是否有未保存的更改，并提示保存
- **保存后自动转换**：选择"保存并转换"后自动执行转换操作
- **Docker 部署支持**：新增 Dockerfile 和 docker-compose.yml
- **PM2 部署支持**：新增 ecosystem.config.js 配置文件

### 界面优化
- **配置管理重构**：将配置列表改为下拉选择形式，界面更简洁
- **按钮布局调整**：配置操作按钮移至下拉框下方，避免遮挡目标模板区域
- **操作说明更新**：更新操作说明文档，反映最新功能

### 问题修复
- 修复删除源文件时报错的问题
- 修复删除第二个文件时 `Cannot read properties of null` 错误
- 修复 `checkConfigHasChanges` 函数变量名拼写错误
- 修复 `removeFile` 和 `drawMappings` 函数的空值检查问题

### 代码优化
- 添加错误处理和空值检查
- 优化 DOM 元素操作的安全性
- 统一使用 `showSuccess` 和 `showError` 替代 `alert`
