# 更新日志

## [1.8.0] - 2026-03-21

### 配置管理功能
- **独立配置管理界面**：新增右上角"⚙️ 配置管理"按钮，无需上传文件即可管理配置
- **选择性导出配置**：支持勾选单个或多个配置进行导出，导出文件包含完整配置信息
- **批量导入配置**：支持导入单个或批量配置文件，智能处理配置冲突
- **冲突处理机制**：导入同名配置时提示选择"覆盖"或"跳过"，避免误操作
- **配置列表详情**：配置管理弹窗中显示配置创建时间、源字段数、目标字段数、映射数等详细信息
- **删除配置功能**：支持在配置管理界面删除配置，删除前弹出确认提示
- **跨浏览器/设备支持**：导出的 JSON 配置文件可在任何浏览器或电脑上导入使用
- **配置按钮精简**：移除配置选择器旁的导入、导出、删除按钮，统一在配置管理界面操作

### 用户体验优化
- **控制面板可拖拽**：底部控制面板新增拖拽手柄，可拖动到任意位置，避免遮挡字段映射
- **位置记忆功能**：控制面板拖动位置自动保存，刷新页面后恢复到上次位置
- **拖拽提示优化**：拖拽手柄显示"🖱️ 点击可拖动控制面板"文字提示
- **配置信息栏布局优化**：配置信息栏移至配置选择器和保存按钮之间，布局更合理
- **控制面板紧凑化**：优化控制面板宽度，自适应内容大小，减少空白区域
- **布局居中对齐**：配置行内容居中对齐，视觉效果更美观

### 界面美化
- **手动映射提示优化**：提示文字分行显示，每句话独立一行，更易阅读
- **图标化提示**：使用蓝色●、红色●、绿色──等图标替代文字描述，更直观
- **第三步闪烁提示**：当有必填字段未映射时，手动映射提示框边缘橙色闪烁，吸引注意
- **舒适闪烁频率**：闪烁频率调整为 3 秒/次，光晕柔和，既醒目又不焦虑
- **颜色突出显示**：提示文字中的"蓝色"、"变红"等关键词使用对应颜色加粗显示

### 问题修复
- **按钮空值错误修复**：修复设置按钮 disabled 属性时的空值检查，避免"Cannot set properties of null"错误
- **保存配置弹窗修复**：修复保存配置确认对话框的异步处理逻辑，确保弹窗正确关闭
- **配置加载优化**：优化配置加载流程，确保配置正确应用并更新 UI 状态

## [1.7.0] - 2026-03-21

### 核心功能优化
- **数据转换逻辑重构**：修复字段映射数据填充逻辑，现在严格按照映射关系将源数据填充到目标列
- **空值处理优化**：源字段为空时，目标字段正确设置为空值，不再使用模板原始值填充
- **模板数据处理优化**：生成结果文件时只使用模板的表头结构，不复制模板的数据行内容
- **配置名称保存修复**：保存配置时正确保存配置名称字段，解决加载配置时显示"undefined"的问题
- **配置文件字段完善**：保存配置时为所有映射关系添加 `sourceField` 和 `targetField` 字段描述

### 用户体验优化
- **保存配置提示优化**：保存配置前检查必填字段映射情况，如有未映射的必填字段先提示确认
- **配置列表自动刷新**：保存配置后自动刷新配置下拉列表并选中刚保存的配置
- **保存并转换功能修复**：修复点击"保存并转换"后配置保存了但没有执行转换的问题
- **输出文件名优化**：转换生成的文件名使用当前配置的名称作为前缀，不再显示"默认配置"
- **数据预览功能修复**：修复源文件数据预览打开后看不到数据的问题
- **字段映射自动建立**：选择目标文件后，如果没有匹配的配置，自动建立字段名相同的映射并提示新建配置

### 问题修复
- **映射分析失败修复**：修复多处 `previewData.mappings` 未进行空值检查导致的报错
- **转换时参数错误修复**：修复 `buildOutputRows` 和 `writeOutputFile` 函数参数顺序错误
- **Workbook 无效错误修复**：修复 `writeOutputFile` 函数缺少返回值导致的"Invalid Workbook"错误
- **必填字段检测优化**：点击"开始转换"时重新计算必填字段映射情况，确保提示准确
- **配置加载空值检查**：添加配置加载时的空值检查，避免加载不存在的配置时报错

## [1.6.0] - 2026-03-19

### 新增功能
- **连线交互优化**：连线现在显示在面板上方，起点和终点不会被遮挡
- **操作提示更新**：更新手动映射提示文字，包含建立映射、删除映射和设置转换规则的完整说明

### 界面优化
- **连线图层调整**：调整 SVG canvas 的 z-index，确保连线始终显示在最上层
- **转换规则指示器清除**：删除映射或删除文件时，自动清除相关的转换规则指示器

### 问题修复
- **必填缺失统计修复**：修复必填缺失字段数量计算逻辑，现在会正确统计已映射的必填字段
- **服务持续运行**：移除生成文件后自动关闭服务的逻辑，服务现在会持续运行
- **连线点击区域修复**：修复连线点击区域问题，现在可以在连线任何位置点击删除
- **⚙按钮点击修复**：修复连线中间设置按钮点击无反应的问题
- **文件删除按钮修复**：修复文件删除按钮被遮挡无法点击的问题

### 字段匹配逻辑优化
- **高置信度**：字段名称完全一样时（去掉 `*` 并 trim 后比较）
- **中置信度**：字段里包含相同含义的词语
- **无匹配**：没有共同词语时不再建立低置信度映射

## [1.5.0] - 2025-03-16

### 新增功能
- **数据预览弹出框**：点击源文件行数可弹出预览窗口，显示前5行数据
- **配置变化检测**：转换前自动检测配置是否有未保存的更改，并提示保存
- **保存后自动转换**：选择"保存并转换"后自动执行转换操作
- **Docker 部署支持**：新增 Dockerfile 和 docker-compose.yml
- **PM2 部署支持**：新增 ecosystem.config.js 配置文件

### 界面优化
- **配置管理重构**：将配置列表改为下拉选择形式，界面更简洁
- **按钮布局调整**：配置操作按钮移至下拉框下方，避免遮挡目标模板区域
- **操作说明更新**：更新操作说明文档，反映最新功能

### 问题修复
- 修复删除源文件时报错的问题
- 修复删除第二个文件时 `Cannot read properties of null` 错误
- 修复 `checkConfigHasChanges` 函数变量名拼写错误
- 修复 `removeFile` 和 `drawMappings` 函数的空值检查问题

### 代码优化
- 添加错误处理和空值检查
- 优化 DOM 元素操作的安全性
- 统一使用 `showSuccess` 和 `showError` 替代 `alert`
