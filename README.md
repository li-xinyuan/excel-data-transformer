# Excel 数据转换工具

一个可视化的 Excel 数据转换工具，支持智能字段映射和值转换规则配置。

## 功能特性

- 🎯 **智能字段映射** - 自动分析源数据和目标模板，建立字段映射关系
- 🔧 **手动调整映射** - 可视化拖拽连线，手动建立或修改映射关系
- 🔄 **值转换规则** - 配置字段值转换规则（如"已完税"→"超出免税部分"）
- 📊 **数据预览** - 查看源数据前5行，方便确认字段内容
- 💾 **配置保存** - 保存映射配置，下次使用直接加载
- 📈 **必填字段检查** - 自动检测缺失的必填字段
- 🎨 **可视化连线** - 清晰的映射连线，支持点击删除和设置转换规则

## 环境要求

- Node.js 14.0 或更高版本
- 支持的操作系统：Windows / macOS / Linux

## 安装步骤

### 1. 安装 Node.js

如果还没有安装 Node.js：

- **Windows**: 访问 https://nodejs.org/ 下载 LTS 版本安装
- **macOS**: 使用 Homebrew 安装 `brew install node`
- **Linux**: 使用包管理器安装，如 `sudo apt install nodejs npm`

### 2. 安装依赖

在项目目录下打开命令行，执行：

```bash
npm install
```

## 使用方法

### 方式一：使用启动脚本

```bash
# Windows
双击运行 start.bat

# macOS / Linux
./start.sh
```

### 方式二：命令行启动

```bash
node transform_excel.js
```

### 方式三：开发模式

```bash
node src/index.js
```

启动后，在浏览器中打开显示的地址（默认 http://localhost:3456），通过界面上传文件。

## 操作说明

1. 将源数据 Excel 文件拖拽到左侧区域（或点击选择文件）
2. 将目标模板 Excel 文件拖拽到右侧区域
3. 系统自动分析并建立字段映射，可手动调整映射关系
4. **手动映射**：先点击源字段蓝色圆点（变红），再点击目标字段圆点即可建立连线
5. **删除映射**：点击连线可删除该映射关系
6. **设置转换规则**：点击连线中间的⚙按钮可配置字段值转换规则
7. 已配置值转换规则的字段会显示🔄标记
8. 确认无误后点击"开始转换"

## 字段匹配规则

- **高置信度（绿色）**：字段名称完全一样（去掉 `*` 并去除首尾空格后比较）
- **中置信度（黄色）**：字段里包含相同含义的词语
- **手动映射（蓝色）**：用户手动建立的映射关系

## 常见问题

**Q: 为什么打开浏览器后显示空白？**
A: 请确保命令行窗口没有关闭，服务正在运行中。

**Q: 转换后的文件保存在哪里？**
A: 默认保存在源数据文件同目录下，文件名添加"_转换结果"后缀。也可以在界面上指定输出路径。

**Q: 支持哪些 Excel 格式？**
A: 支持 .xlsx 和 .xls 格式。

**Q: 如何删除映射关系？**
A: 点击连线即可删除该映射关系。

**Q: 如何设置字段值转换规则？**
A: 点击连线中间的⚙按钮，在弹出的窗口中添加转换规则。
